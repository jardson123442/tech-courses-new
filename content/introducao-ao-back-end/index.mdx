# Introdução ao Back End

# O que é back-end?

Como o próprio nome sugere, vem da ideia daquilo que tem por trás de uma aplicação. Pode ficar meio abstrato num primeiro momento, mas pense que para conseguir usar o Facebook no dia a dia, os dados (as informações) do seu perfil, amigos e publicações precisam estar salvos em algum lugar e serem processados a partir dele, sendo este lugar um banco de dados.(“O que é back-end guia completo”, Harve, Escola de Tecnologia)

O Back-End trabalha em boa parte dos casos fazendo a ponte entre os dados que vem do navegador rumo ao banco de dados e vice-versa, sempre aplicando as devidas regras de negócio, validações e garantias num ambiente restrito ao usuário final (ou seja, onde ele não consegue acessar ou manipular algo).(SOUTO, Mario, Alura)

O back-end lida com a lógica do servidor. Em uma estrutura web, o servidor é quem responde às requisições de clientes e oferece o que foi solicitado. Nesse viés, auxilia a gerenciar também a conexão com os bancos de dados, a questão da segurança e conexão com outras aplicações, por meio de APIs (um código em comum que possibilita a comunicação entre dois sistemas.

Em termos práticos, utilizando o exemplo do formulário de login, o back-end recebe os dados que foram digitados e enviados via rede. Então, ele os processa e verifica se aquelas informações constam no banco de dados da aplicação. Caso estejam, o back responde com uma confirmação e permite o acesso.(Redação Tera)

# As linguagens mais populares de programação back-end:

1. Java:
   - Uma linguagem orientada a objetos, é conhecida por sua portabilidade e amplamente usada em desenvolvimento corporativo.
   - Utilizada para desenvolver aplicativos móveis (Android), sistemas empresariais e aplicativos web.
   - A 3º linguagem de programação mais popular do mundo.
   - É uma linguagem mais difícil para iniciantes do que as outras linguagens backend.
2. PHP:

   - Uma linguagem de script do lado do servidor usada principalmente para desenvolvimento web.
   - Ênfase em integração com bancos de dados e interatividade dinâmica nas páginas da web.
   - Muitas vezes usada em conjunto com sistemas de gerenciamento de conteúdo (CMS) como WordPress e Joomla.
   - Alimenta 78.2% dos websites.
   - Tolera mais erros

3. .NET:

   - Uma plataforma de desenvolvimento da Microsoft que suporta várias linguagens, incluindo C# e Visual Basic .NET (VB.NET).
   - Amplamente utilizada para criar aplicativos Windows, web e móveis, além de jogos e aplicativos empresariais.
   - Altamente estável e confiável

4. Ruby:

   - Uma linguagem de script dinâmica e orientada a objetos, conhecida por sua simplicidade e produtividade.
   - Muito utilizada no desenvolvimento web e aplicativos web, especialmente com o framework Ruby on Rails.
   - Cresceu em popularidade no começo dos anos 2000, mas vem caindo desde então.
   - É fantástico para criar protótipos.

5. Python:

   - Uma linguagem de programação de alto nível, versátil e fácil de aprender.
   - Amplamente utilizada em ciência de dados, inteligência artificial, automação, desenvolvimento web e muito mais.
   - Ótimo para iniciantes.
   - A linguagem de programação que mais cresce.

6. JavaScript:

   - Uma linguagem de programação essencial para desenvolvimento web, permitindo interatividade e dinamismo nas páginas da web.
   - Utilizada tanto no lado do cliente (navegador) quanto no lado do servidor (Node.js).
   - Pode ser utilizada tanto para front e back-end.
   - Muito popular com uma grande comunidade.
   - Pode ser difícil de ser mantida e escalada.

7. TypeScript:

   - Uma extensão do JavaScript que adiciona tipagem estática ao código, facilitando o desenvolvimento e a manutenção de aplicativos complexos.
   - Amplamente adotada em projetos de grande escala, especialmente em aplicações web.
   - É a 7º linguagem de programação mais utilizada
   - Desenvolvida pela Microsoft para facilitar a programação do JavaScript

8. Go (Golang):
   - Uma linguagem de programação compilada e de código aberto criada pelo Google, conhecida por sua eficiência e concorrência.
   - Usada para desenvolver aplicativos de servidor, sistemas de backend, ferramentas de linha de comando e muito mais.
   - Sintaxe similar ao C/C++.

Cada uma dessas linguagens possui suas próprias características, pontos fortes e áreas de aplicação, sendo escolhida com base nos requisitos do projeto e na preferência da equipe de desenvolvimento.

# O que faz um desenvolvedor back-end?

Este é o profissional que atua nos “bastidores”, por assim dizer. Enquanto o profissional front-end trabalha na parte visível de um projeto, ou seja, na interface de usuário, o dev back-end trabalha na parte invisível para o visitante do site, usuário do aplicativo ou programa de computador. (CROWD)

O desenvolvedor back-end também lida com a lógica da aplicação, implementando algoritmos e estruturas de dados que possibilitam o funcionamento correto das funcionalidades. Eles se preocupam com a eficiência, segurança e escalabilidade da aplicação, garantindo que ela possa lidar com um grande volume de usuários e dados.(CROWD)

Além disso, o desenvolvedor back-end trabalha na integração de diferentes sistemas e APIs (interfaces de programação de aplicativos). Eles garantem que a aplicação se comunique eficientemente com outros serviços, sistemas de pagamento, serviços de terceiros, entre outros, para oferecer uma experiência completa e funcional ao usuário.(CROWD)

É importante lembrar que, apesar de estar nos bastidores, este profissional também é muito importante para garantir uma boa experiência de uso das plataformas. Portanto, ele também deve trabalhar pensando em soluções para que o uso da aplicação seja rápido, preciso etc.(Redação Tera)
