# Introdução ao Desenvolvimento de Jogos

# GAMES - O que esperar?

A carreira de desenvolvimento de jogos é uma indústria impactante e em constante crescimento que envolve a criação de jogos eletrônicos para diversas plataformas, como consoles, PCs, dispositivos móveis e realidade virtual.

Segundo a Casa do Desenvolvedor, a indústria de jogos eletrônicos tem experimentado um crescimento exponencial nos últimos anos, com cada vez mais pessoas se interessando por jogos, a demanda por profissionais qualificados nessa área também tem aumentado significativamente (Casa do Desenvolvedor, 2023, p. 1)

Segundo a Casa do Desenvolvedor (2023), os desenvolvedores de jogos trabalham em equipes multidisciplinares para criar experiências interativas que envolvem os jogadores em narrativas emocionantes, desafios criativos e mundos virtuais únicos. De acordo com a mesma fonte, para se tornar um desenvolvedor de jogos é essencial ter conhecimentos sólidos de programação, principalmente em linguagens como C++, Java e Python, que são amplamente utilizadas na criação de jogos.

Vamos dar uma leve introdução sobre a carreira de desenvolvimento de jogos. A criação de jogos é uma das ramificações da profissão de um desenvolvedor, a criação de jogos para celular, computador, Xbox, entre outros meios. O desenvolvimento de jogos tem algumas etapas, que seria o pré-requisito para começar na carreira.

## **Impacto da área**

A área de desenvolvimento de jogos tem um impacto significativo na sociedade, que vai além do entretenimento. Jogos eletrônicos são uma forma popular de entretenimento em todo o mundo. Eles moldam a cultura popular e influenciam a maneira como as pessoas se divertem. (Casa do Desenvolvedor, 2023, p. 2)

Jogos educacionais são usados para ensinar uma variedade de habilidades, desde matemática até línguas estrangeiras. Simulações e jogos são usados em treinamento para profissionais em áreas como medicina, aviação e militares. Jogos também podem desenvolver habilidades como resolução de problemas, tomada de decisões, coordenação motora, raciocínio lógico e habilidades sociais(Supera, 2023).

Jogos de dobra de proteínas, por exemplo, são usados para pesquisas científicas e médicas. Os jogadores podem contribuir para a pesquisa enquanto se divertem jogando. Jogos estão sendo projetados para serem jogados por pessoas com diversas habilidades e necessidades, tornando o jogo acessível para todos os públicos. (Profissionais Biotec, 2023).

O desenvolvimento de jogos tem impulsionado a pesquisa e a adoção de tecnologias de realidade virtual e aumentada, que têm aplicações além dos jogos, como treinamento, simulações e experiências imersivas. (Bard, 2023)

## **Trilha e visão profissional**

O trabalho na área de desenvolvimento de jogos é altamente colaborativo e envolve diversas disciplinas para criar jogos eletrônicos. Aqui está uma visão geral de como funciona a trilha nessa área, segundo Oliveira (2023):

**Etapa 1:**

Primeiro passo é aprender **lógica** com alguma linguagem de programação, a maioria dos jogos utilizam as C++, C#, Java ou Python, para criação de jogos.

Segundo passo é **designer de jogos**, nesta etapa irá ser escolhido as mecânicas do jogo, níveis, personagens e narrativa.

Terceiro passo é **arte e design**: Para criar gráficos, animações e interfaces de usuário atraentes.

Quarto passo é **música e efeitos sonoros**, todo bom jogo precisa de uma trilha sonora e efeitos sonoros para dar emoção a interação no jogo.

**Etapa 2:**

Escolha uma **plataforma**, existem várias plataformas de jogos PC, consoles (como PlayStation, Xbox, Nintendo), dispositivos móveis (iOS e Android) ou realidade virtual. Tendo essa gama de plataformas a seguir, você pode escolher a que melhor se encaixar no seu gosto pessoal ou profissional.

**Etapa 3:**

Escolha uma **engine de desenvolvimen**to como Unity, Unreal Engine, Godot e outras que oferecem muitas possibilidades para criação dos jogos, tem-se essa diversidade de ferramentas poderosas para auxiliar nessa caminhada.

**Segue abaixo um vídeo sobre desenvolvimento de jogos:**

[https://youtu.be/WT33e8ewuOA?si=7bc7HjECNyf-hPLr](https://youtu.be/WT33e8ewuOA?si=7bc7HjECNyf-hPLr)

**Etapa 4:**

Criar o c**onceito do jogo,** desenvolvimento uma ideia clara do que ocorrerá no jogo, com mecânicas de jogo, narrativa, personagens e a estética do jogo. Existe o GDD, que é um documento de design de jogos que auxilia na organização das ideias.

**Etapa 5:**

**Desenvolvimento** é nessa etapa que o jogo começa a ser criado, é feita a programação, criação de arte, música e efeitos sonoros. Os testes têm que ser feitos rotineiramente, a cada modificação tem que haver o teste para que tudo funcione dentro dos conformes.

**Etapa 6:**

**Teste e interação** é nessa fase que outras pessoas irão testar o jogo, e dar o feedback sobre suas percepções, e as melhorias futuras devem ser feitas com base nesse feedback do usuário.

**Etapa 7:**

**Polimento** é a parte que haverá o refinamento e otimização do jogo para garantir a jogabilidade e a eliminação de erros.

**Etapa 8:**

Prepare-se para o **lançamento**, nesta etapa o jogo será lançado na plataforma escolhida, pode ser necessário a criação de um site, a configuração de contas de desenvolvedor e a promoção do jogo.

**Etapa 9:**

**Marketing** e **distribuição** do jogo, essa é a parte que os jogadores têm que ser atraído para o seu jogo através das mídias sociais, blogs, canais de vídeo entre outros métodos, a estratégia de marketing fica a critério dos responsáveis pelo jogo.

**Etapa 10:**

O **Suporte pós lançamento** é a etapa de melhoria contínua, tem que estar sempre a frente atualizando e melhorando seu jogo com base no feedback dos jogadores, também pensando em conteúdos novos e correções de bugs.

## **Códigos**

Vamos demonstrar alguns pequenos exemplos de códigos que irão auxiliar na criação de jogos.

A plataforma Unity é atualmente uma das mais utilizadas no desenvolvimento de jogos mundialmente. Podendo criar desde jogos 2d até jogos mais complexos 3d para PC, console e mobile. Alguns dos imports para um desenvolvedor de jogos em Java segundo a Oracle (2023):

- **import javafx.application.Application:** Este import permite que você use a classe Application, que é a classe base para criar aplicativos JavaFX. Geralmente, você cria uma classe que estende Application e implementa o método start(Stage primaryStage) para iniciar seu aplicativo JavaFX.

- **import javafx.scene.Group:** A classe Group é usada para agrupar outros nós gráficos em uma cena (classe Scene). Ela permite criar hierarquias de elementos gráficos e organizá-los dentro da cena.

- **import javafx.scene.canvas.Canvas;:** O Canvas é um componente gráfico que fornece um espaço de desenho onde você pode renderizar gráficos 2D. Você pode obter o contexto de desenho do Canvas usando a classe Graphics Context para desenhar formas, imagens e texto.

- **import javafx.scene.canvas.GraphicsContext;:** A classe GraphicsContext é usada para realizar operações de desenho, como desenhar retângulos, linhas, imagens e texto em um Canvas. Ela fornece métodos para controlar a aparência e o posicionamento dos elementos gráficos.

- **import javafx.stage.Stage;:** A classe Stage representa a janela principal de um aplicativo JavaFX. Ela é necessária para iniciar a interface do usuário e é geralmente passada como argumento para o método start(Stage primaryStage).

Esses imports são essenciais ao criar aplicativos JavaFX, pois fornecem as classes e recursos necessários para criar e gerenciar a interface do usuário e a renderização gráfica Oracle (2023).

Além dos imports que são usados para codar em um projeto de jogos, temos também exemplos do código na prática. Abaixo estão alguns códigos usados em jogos:

### **Desenhando um Retângulo na Tela:**

```java
gc.setFill(Color.RED);

gc.fillRect(x, y, largura, altura);
```

### **Movimentando um Objeto na Tela:**

```java
scene.setOnKeyPressed(event -> {

	if (event.getCode() == KeyCode.LEFT) {

		x -= velocidade;

	} else if (event.getCode() == KeyCode.RIGHT) {

	} else if (event.getCode() == KeyCode.UP) {

		y -= velocidade;

	} else if (event.getCode() == KeyCode.DOWN) {

		y += velocidade;

	}

});
```

## **Mercado**

A indústria de jogos não se limita apenas a programadores. Ela abrange diversos outros profissionais, incluindo artistas de jogos, designers, escritores, profissionais de testes, engenheiros de som, produtores, entre outros profissionais. O mercado de jogos está em crescimento constante por aumentar a demanda anualmente, os jogos são uma grande fonte de entretenimento interativo e isso é muito atrativo para os usuários (Associação Brasileira do Setor de Jogos Eletrônicos, 2023).

O mercado é composto por muitas startups e estúdios independentes que oferecem oportunidades de emprego na área de desenvolvimento. Para se dar bem na área é importante ter habilidade técnica em programação, design gráfico, modelagem 3D, animação, escrita criativa e design de níveis (Associação Brasileira do Setor de Jogos Eletrônicos, 2023).

Um grande atrativo para na área de jogos além das oportunidades, é o salário. Um programador de jogos ganha em média R$8.290,00/mês. Quem deseja seguir essa carreira deve se empenhar nos estudos, o mercado é superaquecido para o bom programador, não só o programador de jogos. Então foque nos estudos, Glassdoor (2023).
